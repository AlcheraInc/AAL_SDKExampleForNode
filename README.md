# AAL_SDKExampleForNode
A project with napi to connect between C++ and Node.js

## Environments
### Current Node.js Version
 - v18.16.0
### Dependencies
 - bindings
 - cmake-js
 - node-addon-api

## Setup
### Install packages with npm
```
npm install
npm update
```

### Build with cmake-js
```
cmake-js --directory ./examples/
```

### Run test
```
[PATH_OF_NODES].node index.js
```
